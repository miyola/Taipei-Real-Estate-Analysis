# 臺北市房地產市場深度分析報告：空間效率與預算熱區研究

## 🚀 專案簡介
本專案利用 **FineBI** 對臺北市實價登錄數據進行多維度分析。透過視覺化儀表板，量化臺北市居住空間的「公設負擔」情況，並結合「預算區段分組」揭示各行政區的市場定位。

## 🛠 關鍵技術與處理流程
*   **數據來源**：[臺北市資料大平臺/臺北市實價周報](https://data.taipei/dataset/detail?id=a9a97996-3a55-46c8-9076-e5ebdefad6dc)
*   **分析週期**：民國 115 年第 16 週 (1150422)
*   **處理重點**：
    *   **指標建模**：計算 `實得坪數` 與 `公設比 (%)`。
    *   **預算分組**：將總價劃分為小資 (<1500萬)、換屋 (1500-3000萬)、高端 (3000-6000萬)、豪宅 (>6000萬) 四大區間。      *   **數據過濾**：排除無效樣本與非住宅交易，提升分析效度。

## 💡 核心洞察 (Key Insights)
*   **公設比趨勢**：北市購屋者平均須支付逾 1/3 的總價於共有空間。
*   **成交熱區**：中山區為本週流通性最高區域，中正區則呈現高端與豪宅產品之價格多樣性。
*   **預算熱區**：大同區提供相對親民的小資入主機會。
